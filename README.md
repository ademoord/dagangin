# Dagangin

POS-ERP micro-web app for Indonesian UMKM. Flask-only stack with HTMX, Alpine.js, and Chart.js.

## Quick start

```bash
pip install -r requirements.txt
python run.py
```

Open http://127.0.0.1:5001 — login: `admin` / `dagangin123`
